# Análisis de Ventas y Comportamiento de Clientes

Proyecto de análisis de datos orientado a normalización, exploración y generación de insights a partir de un dataset de ventas.

## Objetivo
Transformar una base de datos de ventas desestructurada en un modelo relacional y analizar el comportamiento de clientes y productos para identificar patrones relevantes para el negocio.

## Dataset
- Ventas de productos
- Variables: cliente, producto, ubicación, método de pago, rating, etc.

## Metodología
1. Normalización del dataset mediante modelado relacional.
2. Creación de vistas para facilitar el análisis.
3. Análisis exploratorio de clientes, productos y ratings.
4. Generación de insights descriptivos.

## Resultados
- El 68% de los clientes son hombres.
- El 52% del volumen se concentra en un número reducido de segmentos.
- En algunas categorías, un solo producto concentra gran parte del ingreso.

## Insights clave
- Existe una alta concentración del negocio en pocos segmentos, lo que permite estrategias focalizadas.
- La dependencia de productos outliers implica riesgo ante cambios de demanda.
- El rating no muestra una relación fuerte con la suscripción, sugiriendo otros factores relevantes.
