# Análisis de Ventas y Comportamiento de Clientes

Proyecto de análisis de datos orientado a normalización, exploración y generación de insights a partir de un retail de venta de productos outlet en EEUU.

## Objetivo
Transformar una base de datos de ventas desestructurada en un modelo relacional y analizar el comportamiento de clientes y productos para identificar patrones relevantes para el negocio.

 ## 📑 Columnas del dataset

- **Customer ID**  
  Identificador único del cliente.

- **Age**  
  Edad del cliente.

- **Gender**  
  Género del cliente.
- **Location**  
  estado donde se encuentra el cliente.

- **Item Purchased**  
  Nombre del producto comprado 

- **Category**  
  Categoría del producto
  
- **Size**  
  Talle del producto .

- **Color**  
  Color del producto.

- **Season**  
  Temporada asociada al producto.

- **Shipping Type**  
  Tipo de envío.

- **Payment Method**  
  Método de pago.

- **Purchase Amount (USD)**  
  Monto de la compra en dólares.

- **Date**  
  Fecha en la que se realizó la compra.

- **Subscription Status**  
  Estado de suscripción del cliente.

- **Promo Code Used**  
  Indica si se usó un código promocional, si el cliente esta suscripto este promo se aplica automaticamente.

- **Discount Applied**  
  Indica si se aplicó un descuento del 10%, dado por si se uso o no el codigo de descuento.

- **Previous Purchases**  
  Número de compras previas del cliente.

- **Base Price**  
  Precio base del producto antes de descuentos.

- **Total Discount %**  
  Porcentaje total de descuento aplicado, siendo 10% si se usa el codigo de descuento y luego un 3% por cada temporada pasada desde el tiempo de compra.

- **Final Price**  
  Precio final después de aplicar descuentos.

- **Review Rating**  
  Calificación del producto dada por el cliente.


## Metodología
1. Normalización del dataset mediante modelado relacional.
2. Creación de vistas para facilitar el análisis.
3. Análisis exploratorio de clientes, productos y ratings.
4. Generación de insights descriptivos.
